HTML

  • !DOCTYPE html lets the browser know to use html
  • The head element contains information about the webpage.
  • title is the title that will appear in the browser
  • The body element represents the visible content shown to the user.
  • h1, h2, h3, h4, h5, h6 are the tags to use for headlines within the main
  • header is the area at the top of the page
  • footer is the area at the bottom of the page
  • ul means unordered list
  • ol means ordered list
  • li is an item in the list

Git

  • git init: initialize project to use git
  • git add .: add all changes that you have made
  • git add *filename*: add single file to be saved
  • git commit -m "message": save changes with message
  • git push origin master: push changes to github master
  • git push origin new-branch: push changes to github new-branch
  • git pull origin master: pull changes from github master
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git log: see all previous saved changes
  • git checkout *commit hash*: travel back to old commit
